### CH02 - 변수의 선언과 저장

 * 변수란?
   : 변하는 수 , 하나의 값을 저장할 수 있는 메모리 공간 ( RAM )

  - 메모리에 값을 다시 저장하면 마지막에 저장한 값으로 바뀐다
  - 변수 선언 이유 : 값을 저장할 공간을 마련(메모리)
  - 선언 방법 : 변수타입 변수이름;     ex ) `int age;`


> 변수 명명 규칙  
> 1. 대소문자가 구분되며 길이에 제한이 없다  
> 2. 예약어를 사용해서는 안된다(true등)  
> 3. 숫자로 시작해서는 안된다  
> 4. 특수문자는 _ 와 $만 허용한다.  


> 명명규칙 권장사항  
> 1. 클래스의 이름 첫글자는 대문자, 변수메서드 첫글자는 소문자  
> 2. 여러단어 이름은 단어의 첫글자를 대문자  
> 3. 상수의 이름은 대문자, 단어는 _로 구분한다(상수의 경우 단어구분시)  




 * 변수에 값 저장
   : 대입을 통해 저장

   - ` = ` 등호가 아니라 대입
   - `int age=25;` 이런식
  


 * 변수의 초기화
   : 변수에 처음으로 값을 저장하는것(값을 넣는거)
  
   - 변수는 클래스변수, 인스턴스변수, 지역변수가있는데 지역변수는 읽기전에 꼭! 초기화 해줘야됨
   - `int x=0, y=5;`로 한줄로 여러개 가능

> `int x=10, y=20;` // 두 값을 바꿀려면 어떻게 해야할까?
> 변수를 하나 더 만들면 됨


### CH02 - 변수의 타입

* 변수의 타입
  : 변수의 타입은 저장할 값의 타입에 의해 결정된다
  : 저장할 값의 타입과 일치하는 타입의 변수를 선언

- 값의 타입 : 문자(char), 숫자(정수(byte,short,int,long), 실수(float,double)), 논리(boolean)
- 8개를 기본형이라고 함

- 참조형 : 기본형을 제외한 나머지로 '메모리주소'를 저장(4byte-32bitJVM, 8byte-64bitJVM)
  `Date today;//참조형 변수 today를 선언
  today = new Date(); //today에 객체의 주소를 저장(객체생성)`

* 기본형의 종류와 크기
  : 논리형, 분자형, 정수형, 실수형

  - 논리형 boolean - true, false중 하나, 조건식이나 논리적 계산
  - 문자형 char- 문자를 저장하는데 사용, 변수당 하나의 문자. 여러문자는(String)
  - 정수형 : 정수값저장. 주로사용하는것 int와 long이며 byte는 이진 데이터를 다루는데 사용
            byte는 이진데이터 다룰때 사용
            short는 c언어와 호환을 위해 추가됨(잘 안쓰임)
  - 실수형 : 실수값 저장할때 사용. float 와 double
 
    |종류\크키|1|2|4|8|
    |:--:|:--:|:--:|:--:|:--:|
    |논리형|boolean||||
    |문자형||char|||
    |정수형|byte|short|int|long|
    |실수형|||float|double|

    int와 double이 기본형

* 기본형의 표현범위
  : 1byte= 8bit로 n 비트로는 2의n제곱개 표현 가능

  - 부호없는 정수의 범위 : 0~2의n제곱-1
  - 부호있는 정수범위 : -2의 n-1제곱 ~ 2의 n-1제곱 -1
    char타입은 글자임으로 마이너스가 필요없음 > 0~2의 16제곱 -1까지
    int타입 대략 20억~-20억

    float 타입의 경우 1.4E-45~3.4E38 > E뒤 숫자는 10의 제곱. 3.4(가수) x 10의 38제곱(지수) 로 나뉨(부호, 가수, 지수로 나뉨)
    오차가 발생할수잇음(2진수로 표현안해서) > 정밀도가 중요! 정밀도는 가수에 의해 결정됨.
    float 타입은 부호(2개) 지수(8) 가수(23)인데 가수를 정교화를 통해 24자리까지 가능> 2의 24제곱은 10의 7제곱과 8제곱 사이임으로 정밀도는 7
    double은 지수(11개) 가수(52)로 정밀도는 15
    double이 기본값

    
* 변수, 상수, 리터럴

  - 변수 : 하나의 값을 저장하기 위한 공간
  - 상수 : 한 번만 값을 저장 가능한 변수 > 변수앞에 'final'을 붙여주면 됨(새로운 값 저장 불가)
  - 리터럴 : 그 자체로 '값'을 의미(기존의 상수같은개념) > 문자도 같음

`int year(변수) = 2014(리터럴);
final int MAX_VALUE(상수)=100(리터럴);`



### CH02 - 리터럴의 접두사와 접미사

* 리터럴의 종류
  - 논리형 false, true(접미사  없음)
  - 정수형 123,0b0101,077,0xFF,100L  (접미사 L - long 타입, 안붙으면 int)
  - 실수형 3.14, 300e8, 1.4f, 0p-1 (접미사 f, d - 두개뿐이라 d는 생략가능 f float, d double)
  - 문자형 'A','1','\n'(줄바꿈, 개행문자) (접미사  없음)
  - 문자열 "ABC", "123", "A", "true" (접미사  없음) A만 있어도 문자열

  - 정수형 리터럴 앞에 붙으면 진법이 바뀜
  - e7의 경우 10의 7제곱이라는뜻. 2.3e4f는 2.3에 10의 4제곱 곱해서 23000.0f와 같음
  - e + 숫자가 붙은경우 실수형임..
 


  0B,0b > 이진수,   
  0X,0x > 16진수  

```java
long l = 10_000_000_000L;//(L생략 불가-큰숫자 100억이라서, _ 표시는 알아보기 쉬우라고 넣음)
long l = 100;// (L생략가능 int표현으로 표현가능한것들은 생략가능)
float f=3.14f;//(float 타입 생략불가능)
double d =3.14d;//(더블이 기본형이라 생략가능)
```

### CH02 - printf를 이용한 출력

 * 형식화된 출력 - printf()
   : 출력 형식 지정가능

    - println의 단점 : 출력형식 지정 불가
        - 실수의 자리수 조절불가(3.3333...)
        - 10진수로만 출력됨 - 8진수, 16진수 넣어도 10진수

    - `System. out. printf("형식", 내용);`
    - "%.2f" : 소수점 둘째자리
    - "%d" : 정수를 10진수로
    - "%X" : 16진수로 출력
    - %~ 등 : "지시자" 라고 부름
  
      |지시자|설명|
      |:--|:--|
      |%b|불리언 형식으로 출력(boolean)|
      |%d|10진(decimal)정수의 형식으로 출력|
      |%o|8진(octal)정수의 형식으로 출력|
      |%x,%X|부동 소수점(floating-point)의 형식으로 출력|
      |%e,%E|지수(exponent)표현식의 형식으로 출력|
      |%c|문자(character)로 출력|
      |%s|문자열(string)로 출력|
      
      > 정수형태 : %d, %o, %x,%X
      > 실수 형태 : %f, %e,%E (%g도 있는데 둘중 간략한 형식으로 표현됨)
      > 문자형태 : %s
      > JAVAAPI>Formatter찾아보면 지시자 많음

- `System.out.printf("age:%d year:%d\n",14,2017);`  :  형식 한꺼번에 지정가능
- `\n`, `%n` : 줄바꿈 표시(%n은 Os안탐)
- 8진수, 16진수 출력시 접두사를 붙일수 있음(0x,0) / 2진수는 Integer.toBinaryString(숫자) 문자열로 출력해야됨%s
- 접두사 붙이는 법 : "%#o", "%#x", "%#X"
- 실수 출력시 맨 앞부터 7자리 정확f, 0이많으면 지수표현 쓰는게 좋음
-  `System.out.printf("d=%14.10f%n", d);` : 전체 14자리 중 소수점 아래가 10자리(d가 소숫점 위 1자리 아래 7자리면 아래는 0으로 위는 공백으로 채움)
-  왼쪽 정렬 : `System.out.printf("[%-5d]%n",10)` : 5자리 정수중 왼쪽정렬 [10   ]
-  문자열도 같음 "[%-20s]%n` : 20글자중 왼쪽정렬
-  문자열의 경우 "[%.8s]%n" 의 경우 앞의 8글자만 출력 [www.code] 뒤가잇어도 이만큼출력
