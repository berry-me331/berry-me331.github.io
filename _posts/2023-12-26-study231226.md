### CH05 - 배열의 초기화

 - 배열의 초기화
   : 배열의 각 요소에 처음으로 값을 저장하는 것

   `int [] score=new int[] { 10,20,30,40,50}`  
   `int [] score={ 10,20,30,40,50}` new int[] 생략가능  

   `int[] = score;   
   score = new int[] {10,20,30,40,50}   
   score = {10,20,30,40,50} // 에러 new int[]생략 불가능`
